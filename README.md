# Cloud Network & Access Control Architecture

## Project Description
This project presents a secure cloud network architecture for internal
applications. The design focuses on protecting private servers using
firewalls, restricted SSH access, and network segregation.

## Key Features
- Private network for internal application servers
- Firewall rules allowing only required ports
- Restricted SSH access for administrators
- Cloud-agnostic secure design

## Technologies & Concepts
- Cloud Network Security
- Access Control
- Firewalls
- SSH
- Private Servers
